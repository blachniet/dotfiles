## bash

On Ubuntu, set terminal color scheme to Solarized: *Edit > Profile Preferences > Colors > Built-in schemes*

## vim

Requires vim with lua support:
    # Ubuntu
    sudo apt-get install vim-gnome
    # Mac
    brew install vim --with-lua
